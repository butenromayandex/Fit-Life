## MKDosc
```shell
pip install mkdocs
pip install mkdocs_puml
pip install mkdocs-plantuml
```
```yaml
plugins:
    - plantuml:
        puml_url: https://www.plantuml.com/plantuml/
```
новый проект
```shell
mkdocs new docs
```
старт проета

```shell
mkdocs serve
```
билд проекта

```shell
mkdocs build
```
